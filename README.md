## Project Overview

Abnormal Person Finder is a comprehensive Android application designed to address the safety and well-being of individuals, especially those vulnerable to becoming lost or disoriented. The app combines advanced image processing using the OpenCV library with community-driven features to provide a robust and supportive platform for guardians and users.

**Key Features:**

1. **Image Processing and Recognition:**
   - Abnormal Person Finder allows guardians to input crucial information and images of individuals they are responsible for. These details are securely stored in the app's database.
   - When a lost person is found, the image processing feature performs a real-time comparison between uploaded images and the database. If a match is found, the app instantly displays the person's information, facilitating quick and efficient reunions.
   - In cases where there's no match, the app provides a clear "Image Not Found" message.

2. **Lost Person Reporting:**
   - Abnormal Person Finder empowers users to proactively report lost individuals to the community.
   - Notifications are sent to all app users, creating a collaborative network actively engaged in helping find the missing person.
   
3. **Found Person Notifications:**
   - When a lost individual is located, the app sends notifications to all users, celebrating the successful reunion.
   - This feature promotes community support and reinforces the effectiveness of the app in locating missing persons.

4. **Community Chat:**
   - The application fosters communication among users through a built-in chat feature.
   - Users can share information, support one another, and exchange updates about ongoing search efforts.

Abnormal Person Finder is more than just a safety app; it's a powerful tool that brings together technology, community involvement, and compassion to enhance the security of vulnerable individuals. With image processing capabilities and active user engagement, it's a testament to the positive impact technology can have in safeguarding and uniting our communities.
